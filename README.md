Chat聊天室
===
 
chat聊天室是一个基于nodejs+websocket+express+socket.io模块构建的
网页聊天室工具。前端视图使用可vue进行数据交互构建使用了animate.css做过度动画效果

功能介绍
---
* 支持表情发送
* 支持键盘回车发送信息
* 支持在线用户统计和用户列表
* 支持在线用户搜索
* 支持在线用户的私聊
* 支持消息声音提示

使用手册
---
1. 下载代码并安装nodejs
2. 使用’npm install‘安装项目依赖模块
3. 使用’node app.js‘启动服务
4. 最后打开浏览器访问 `http://localhost:3000`
